## Novedades de v0.0.9

- Aviso de nueva version dentro de la app: al iniciar LibreTracks se comprueba si hay una version mas reciente y se muestran las novedades en el idioma de la app, con accesos directos para descargar, recordar mas tarde u omitir la version. Tambien hay un boton "Buscar actualizaciones" en Configuracion - General.
- Mas canales de salida y mejor flujo de seleccion: ahora la salida puede usar mas canales de hardware de forma generica, con un flujo Aplicar/Descartar para evitar reaperturas innecesarias del dispositivo. La seleccion masiva de canales es mas rapida y el selector de ruta de pista usa un combobox con scroll para listas largas.
- Carga de proyectos mas rapida y con menos consumo de RAM: el motor de audio reutiliza la cache PCM entre sesiones cuando el archivo no ha cambiado, transmite los archivos en formato nativo sin pasar por la cache cuando es posible, mantiene la cache fuera de %TEMP% con un limite por LRU y reserva por defecto hasta un 10% del disco libre (minimo 4 GiB). El resultado: importacion y apertura de proyectos grandes notablemente mas agiles.
- Medidores de pista con peak hold: los track meters mantienen visible el pico maximo unos instantes, en escritorio y en el remote, para ver picos breves sin perderselos.
- Tap tempo en el transporte: ahora se puede fijar el BPM base tocando el pulso directamente desde la barra superior.
- Mejoras de integracion con el navegador y de la web: enlaces de la app abren el navegador del sistema con permisos mas afinados; la pagina de descargas incluye una seccion de apoyo voluntario en Ko-fi.

## What's New in v0.0.9

- In-app update notifications: on startup LibreTracks checks for a newer release and shows the changelog in the active app language, with shortcuts to download, snooze, or skip the version. A manual "Check for updates" button is also available in Settings - General.
- More output channels and a smoother selection flow: hardware output now supports more channels generically, with an Apply/Discard flow that avoids unnecessary device reopens. Bulk channel selection is faster and the track routing picker uses a scrollable combobox for long device lists.
- Faster, lighter project loading: the audio engine reuses the PCM cache across sessions when the source file is unchanged, streams native-format files in place without going through the PCM cache when possible, keeps the cache outside %TEMP% with an LRU size cap, and grows the default budget up to 10% of free disk (minimum 4 GiB). Importing and opening large projects feels noticeably snappier.
- Peak hold on track meters: track meters now keep the latest peak visible for a moment, both on desktop and on the remote, so brief peaks are easier to spot.
- Tap tempo in the transport: the base BPM can be set by tapping the pulse directly from the top bar.
- Better browser integration and website polish: in-app links open the system browser through tightened permissions; the downloads page includes an optional Ko-fi support section.
