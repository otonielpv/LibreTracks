## Novedades de v1.4.2

- 📊 Nuevo medidor de recursos en la barra superior: ahora puedes ver de un vistazo el uso de CPU y memoria mientras trabajas, para detectar cuándo una sesión grande empieza a exigir demasiado al equipo.
- 🎧 Corregido el tono al cambiar de dispositivo de audio: si usabas o alternabas entre dispositivos con distinta frecuencia (por ejemplo unos auriculares a 44.1 kHz y otros a 48 kHz), la reproducción podía oírse acelerada o más grave. Ahora suena correcta en todos los casos.
- 🌐 La página de descargas ahora muestra todas las versiones: por defecto aparece la última, pero puedes elegir cualquier versión anterior y descargar sus instaladores sin salir de la web.

## What's New in v1.4.2

- 📊 New resource meter in the top bar: you can now see CPU and memory usage at a glance while you work, so you can tell when a large session starts to push your machine.
- 🎧 Fixed pitch when switching audio devices: if you used or alternated between devices with different sample rates (e.g. headphones at 44.1 kHz and others at 48 kHz), playback could sound sped up or lower in pitch. It now plays back correctly in every case.
- 🌐 The downloads page now lists every version: the latest one shows by default, but you can pick any earlier version and grab its installers without leaving the site.
